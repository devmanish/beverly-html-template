## Swiper js options
```
var swiperTestimonial = new Swiper("#testimonialSlider", {
  slidesPerView: 1,
  spaceBetween: 30,
  pagination: {
    el: ".swiper-pagination",
    clickable: true,
  },
  breakpoints: {
    810:{
      slidesPerView: 2,
    }
  }
});
```
